## 📌 Overview

Customer churn is a major challenge for subscription-based businesses.

**ChurnSense** is an end-to-end Machine Learning system that:

- 📊 Analyzes customer behavior  
- 🤖 Trains multiple ML models  
- 🏆 Selects the best-performing model  
- 💡 Explains churn predictions with reasons  

---

## ✨ Features

- 📊 **Interactive Dashboard**
- 🤖 **Dual Model System**
  - Logistic Regression (baseline)
  - Random Forest (advanced)
- 🏆 **Automatic Best Model Selection**
- 📈 **EDA Visual Insights**
- 💡 **Explainable Predictions**
- ⚡ **Real-time Prediction Interface**

## ⚙️ How It Works

### 📊 Data Processing
- Handles missing values  
- Encodes categorical features  
- Scales numerical data  

---

### 🤖 Model Training
- Trains:
  - Logistic Regression  
  - Random Forest  
- Evaluates both models  
- Selects the **best-performing model automatically**

---

### 🎯 Prediction System
- Inputs:
  - Tenure  
  - Monthly Charges  
  - Contract Type  
  - Internet Service  
  - Gender  
- Outputs:
  - ✅ Churn / Stay  
  - 📊 Probability  
  - 💡 Possible Reasons  

---

## 📊 Key Insights

- 📉 Low tenure customers churn more  
- 💰 High charges increase churn risk  
- 📄 Month-to-month contracts → highest churn  
- 🛠 No tech support → higher churn
